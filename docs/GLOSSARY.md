# Glossary

- **Scenario**: A path (bull / base / bear). Scenarios are not predictions; they are coherent storylines with associated probability weights.
- **Evidence**: Weekly-computed indicator states (e.g. SPY trend, real yields, credit spreads, volatility regime, USD proxy) used as inputs to scoring.
- **Alignment**: Whether observed price (BTC/SPY) falls inside or outside the forecast bands for a given scenario.
- **Forecast version**: Immutable snapshot of band config and meta. New config = new version.
- **Week ending**: Friday (Chicago) that closes the week; used for weekly snapshots and evidence.
- **Nerd Mode**: Opt-in UI that reveals Runs, Admin, Data Health, and Receipts. Persisted via cookie.
